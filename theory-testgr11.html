Section A: Multiple Choice

  1. What is the main function of the CPU?

    2. Memo: The CPU performs the primary computations and processes data.

  2. Which type of software is used to perform specific tasks?

    3. Memo: Application software is used to perform specific tasks.

  3. Which of the following is an example of an input device?

    4. Memo: Examples of input devices include keyboards, mice, and scanners.

Section B: Short Answer

  1. Explain what a file system is.

    2. Memo: A file system manages and organizes files on storage devices, allowing for efficient data access and storage.

  2. What is the difference between primary and secondary storage?

    3. Memo: Primary storage (RAM) is used for temporary storage while the computer is running, whereas secondary storage (HDD/SSD) is used for long-term storage of data and files.

  3. What are the advantages of using cloud storage?

    4. Memo: Advantages of cloud storage include accessibility from anywhere, reduced physical storage needs, and easy data backup and recovery.

Section C: Long Answer

  1. Discuss the impact of technology on education.

    2. Memo: Technology has transformed education by enabling online learning, access to vast resources, and interactive tools for students and teachers.

  2. Describe the steps involved in the information processing cycle.

    3. Memo: The information processing cycle involves input, processing, storage, and output.

  3. Compare and contrast different types of computer networks.

    4. Memo: Different types of computer networks include LAN (Local Area Network), WAN (Wide Area Network), and MAN (Metropolitan Area Network), each serving different geographic areas and purposes.
