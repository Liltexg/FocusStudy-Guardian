Database Management Basics - Grade 11

1. Introduction to Databases

Databases are organized collections of data that are used to manage and store information. Understanding the basics of databases is crucial for data management in various applications.

2. Types of Databases

  • Relational Databases: Use tables to store data and relationships between data.
  • NoSQL Databases: Handle unstructured data and are designed for large-scale data storage.
  • Distributed Databases: Data is stored across multiple locations or systems.

3. Database Management Systems (DBMS)

A DBMS is software that allows users to define, create, maintain, and control access to the database. Common DBMS include MySQL, PostgreSQL, Oracle, and MongoDB.

4. Key Concepts in Database Management

  • Tables: Structures that organize data into rows and columns.
  • Primary Key: A unique identifier for each record in a table.
  • Foreign Key: A field in a table that links to the primary key of another table.
  • SQL (Structured Query Language): A standard language for managing and manipulating databases.
