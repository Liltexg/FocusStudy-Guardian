Section A: Multiple Choice

  1. What is the primary purpose of virtual memory in a computer system?

    2. Memo: Virtual memory allows a computer to compensate for shortages of physical memory by temporarily transferring data from RAM to disk storage.

  2. Which of the following is a characteristic of RISC (Reduced Instruction Set Computer) architecture?

    3. Memo: RISC architecture uses a small, highly optimized set of instructions.

  3. In the context of databases, what is normalization?

    4. Memo: Normalization is the process of organizing data to reduce redundancy and improve data integrity.

Section B: Short Answer

  1. Explain the concept of a class in object-oriented programming (OOP).

    2. Memo: A class in OOP is a blueprint for creating objects, providing initial values for state (member variables or attributes), and implementations of behavior (member functions or methods).

  2. What is the difference between a compiler and an interpreter?

    3. Memo: A compiler translates the entire source code into machine code before execution, while an interpreter translates and executes the code line by line.

  3. Describe the process of data mining and its importance.

    4. Memo: Data mining involves extracting useful information and patterns from large datasets. It is important for making data-driven decisions in various fields such as marketing, finance, and healthcare.

Section C: Long Answer

  1. Discuss the impact of quantum computing on cryptography.

    2. Memo: Quantum computing could significantly impact cryptography by breaking many current encryption algorithms through quantum algorithms like Shor’s algorithm, which can factorize large numbers exponentially faster than classical algorithms. This poses a threat to data security and necessitates the development of quantum-resistant cryptographic methods.

  2. Analyze the ethical implications of genetic engineering.

    3. Memo: The ethical implications of genetic engineering include concerns about playing God, potential harm to the environment and human health, and socio-economic disparities that could arise from access to genetic modifications.

  3. Evaluate the role of cybersecurity in protecting personal and organizational data.

    4. Memo: Cybersecurity plays a crucial role in protecting personal and organizational data from breaches, hacking, and theft. It involves implementing measures like firewalls, encryption, and secure authentication methods to safeguard sensitive information.
