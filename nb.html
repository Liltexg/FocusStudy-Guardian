Network Basics - Grade 12 CAT

1. Networks

Networks are systems that connect multiple devices to share resources and information. Key types of networks include:

  • LAN (Local Area Network): Connects devices within a limited area such as a home or office.
  • WAN (Wide Area Network): Connects devices over large geographic areas, such as between cities or countries.
  • MAN (Metropolitan Area Network): Covers a larger area than a LAN but is smaller than a WAN, such as a city or campus network.
  • VPN (Virtual Private Network): Provides a secure connection over a public network by encrypting data.

2. Intranet vs Internet

The key difference between intranet and internet:

  • Intranet: A private network used within an organization to share information and resources among its members.
  • Internet: A global network connecting millions of private, public, academic, business, and government networks worldwide.

3. Basic Network Security

Network security involves protecting data during transmission and preventing unauthorized access:

  • Firewalls: Control the incoming and outgoing network traffic based on security rules.
  • Encryption: Protects data by encoding it so that only authorized users can decode and access it.
  • Antivirus Software: Detects and removes malware that could compromise network security.
  • Strong Passwords: Use complex and unique passwords for accessing network resources.
  • Regular Updates: Keep software and systems updated to protect against known vulnerabilities.

4. Network Devices

Common devices used in networks:

  • Router: Directs data packets between devices and the internet or other networks.
  • Switch: Connects multiple devices within a LAN and directs data to the appropriate device.
  • Hub: Connects multiple devices in a network but does not direct data like a switch.
  • Modem: Modulates and demodulates data signals for internet connectivity.

5. Network Topologies

Network topology refers to the arrangement of devices and cables in a network:

  • Star Topology: All devices are connected to a central hub or switch.
  • Bus Topology: All devices are connected to a single central cable.
  • Ring Topology: Devices are connected in a circular fashion, with each device connected to two other devices.
  • Mesh Topology: Devices are interconnected, with multiple paths for data to travel.

6. IP Addressing

IP addresses identify devices on a network:

  • IP Address: A unique identifier assigned to each device on a network.
  • IPv4: Uses 32-bit addresses, commonly written as four octets separated by periods (e.g., 192.168.0.1).
  • IPv6: Uses 128-bit addresses to provide a larger address space, written as eight groups of four hexadecimal digits.

7. Network Protocols

Protocols define the rules for data communication over a network:

  • TCP/IP (Transmission Control Protocol/Internet Protocol): The foundational protocol suite for internet communication.
  • HTTP/HTTPS (Hypertext Transfer Protocol/Secure): Protocols for transferring web pages and secure communications.
  • FTP (File Transfer Protocol): Used for transferring files between devices on a network.

8. Troubleshooting Network Issues

Common network problems and solutions:

  • Connection Issues: Check cables, network devices, and connection settings.
  • Slow Performance: Verify bandwidth usage and network congestion.
  • Access Problems: Ensure proper permissions and configuration settings.
  • Device Conflicts: Resolve conflicts with IP addresses and network devices.
